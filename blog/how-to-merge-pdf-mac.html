How to Merge PDF Files on Mac — 4 Free Methods

Published: 2026-06-16 · PDF Tips

Introduction

Mac users have a significant advantage when it comes to PDF management — macOS comes with built-in tools that handle many PDF tasks without requiring any additional software. The Preview app, which ships with every Mac, is far more powerful than most users realize. Beyond viewing images and PDFs, Preview can merge documents, rearrange pages, annotate, and even add signatures. For years, Mac users assumed they needed Adobe Acrobat or expensive third-party software to combine PDFs, but the truth is that macOS provides multiple free methods right out of the box. Whether you are running the latest macOS Sonoma, the widely-used Ventura, or even older versions like Monterey, these techniques work reliably. This article covers four completely free methods to merge PDFs on your Mac, ranging from the quickest drag-and-drop approach to more advanced techniques using Automator and online tools.

Method 1: Using Preview (Built-in macOS App)

Preview is the easiest and most reliable method for merging PDFs on a Mac. Start by opening the first PDF in Preview. Then, make sure the thumbnail sidebar is visible — if not, go to View > Thumbnails or press Option+Command+2. With the sidebar open, you can simply drag additional PDF files from Finder directly into the thumbnail area. Drop them at the position where you want the pages inserted. You can also rearrange pages by dragging thumbnails up or down within the sidebar. If you want to delete certain pages, select the thumbnail and press the Delete key. To save the merged document, go to File > Export as PDF, give it a name, choose a location, and click Save. Note that Preview does not have a dedicated 'Merge' button — the drag-and-drop approach effectively combines the files. This method preserves all original formatting, images, and text, and works on every version of macOS from the past decade.

Additional Tips & Best Practices

Regardless of which method you choose, there are a few universal best practices that will make your PDF workflow smoother. First, always keep a backup of your original files before merging or processing. While most tools are reliable, having a safety net gives you peace of mind. Second, name your files clearly before combining them — this saves time when you need to locate specific documents later. Third, if you are working with scanned documents, consider running OCR (Optical Character Recognition) first to ensure the text remains searchable after merging. Fourth, be mindful of file size limits if you plan to email the merged PDF — some email providers cap attachments at 25MB. Finally, test the merged PDF by opening it on at least two different devices or PDF viewers to ensure compatibility across platforms.

Frequently Asked Questions

Can I merge PDFs on Mac without installing anything?

Yes. Use either Preview (built into macOS), Finder Quick Actions, or a browser-based tool like PDF Craft. All three methods require zero installation.

Does Preview work on older macOS versions?

Yes. Preview's PDF merge capability has been available since at least macOS 10.6 Snow Leopard. The interface may look slightly different on older versions, but the drag-and-drop functionality is identical.

What if the merged file size is too large?

After merging, use a PDF compression tool to reduce the file size. PDF Craft offers a free online compressor that reduces file size by up to 80% while maintaining visual quality.

Can I merge PDFs on Mac using Terminal?

Yes. If you have Python installed, you can use the PyPDF2 library. A simpler option is to use a browser-based tool to avoid command-line complexity.
